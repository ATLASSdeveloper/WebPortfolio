Simulador de Criminalística

Entorno virtual para el entrenamiento de peritos de la Unidad de Criminalística de la Dirección Nacional de Investigación Técnica Científica Policial (DINITEC) en técnicas de inspección ocular técnica (IOT). Este proyecto utiliza tecnologías avanzadas como Godot Engine, Meta Quest 3 y Blender para simular escenarios realistas y dinámicos. Los principales beneficiarios directos son los peritos de la DINITEC, mientras que los beneficiarios indirectos incluyen los cadetes de tercer año de la Escuela Superior de la Policía Nacional y los estudiantes de Criminalística del Instituto Superior Universitario Policía Nacional (ISUPOL).

Gourmet Go Móvil

Versión móvil de Gourmet Go, un sistema complementario al sistema web que permite a los usuarios realizar pedidos de manera sencilla a restaurantes registrados. La aplicación está construida con React para ofrecer una interfaz intuitiva y fluida, y Node.js en el backend para garantizar un rendimiento óptimo. Además, integra una pasarela de pago con PayPal, brindando a los usuarios una experiencia de compra segura y conveniente.

Ver Github

Gourmet Go Web

Versión web de Gourmet Go, un sistema integral diseñado para la gestión eficiente de restaurantes. La plataforma permite administrar el menú ofrecido, la plantilla de empleados y los pedidos realizados a través de la aplicación móvil. Además, para que el restaurante sea visible en la aplicación móvil, es necesario activar una licencia, asegurando un control centralizado y personalizado del servicio.

Reconocimiento de Color y Ropa

Sistema de reconocimiento personal que identifica a una persona, segmenta su ropa y predice el color de las prendas. Los modelos utilizados para estas tareas están alojados en una API que implementa técnicas como redes neuronales convolucionales (CNN), algoritmos K-Nearest Neighbors (KNN) o máquinas de soporte vectorial (SVM), dependiendo del caso, para garantizar flexibilidad y precisión en los resultados.
